# Changelog

All notable changes to the Verriflo Classroom SDK will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-12-15

### Added
- Initial release of Verriflo Classroom SDK
- `VerrifloPlayer` widget for live video streaming
- Adaptive streaming with automatic quality adjustment
- Manual quality control (1080p / 720p / 480p / 360p)
- YouTube-style quality selector popup
- Fullscreen mode with landscape rotation
- Event system for connection lifecycle management
- Overlay support for custom UI elements
- Comprehensive documentation and examples

### Platform Support
- Android (API 21+)
- iOS (12.0+)
- Web (Chrome, Firefox, Safari, Edge)
- macOS (10.14+)
- Windows (10+)
- Linux
